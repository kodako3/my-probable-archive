# my-probable-archive

# What is?
Mission 4GB is a collection of entertainment, knowledge, internet and utility into a single physical unit like a pendrive, sdcard, or any store device. Under 4GB.

# What inside?
- 📚 E-books: A collection of classics, technical manuals, and reference guides.
- 🎮 Games: Lightweight, DRM-free, or portable titles (Indie, Retro, or Terminal-based).
- 🌐 Small Portable Internet: Offline-compatible browser tools, HTML snapshots, or lightweight proxies.
- 🖥 Portable Apps: Essential software (text editors, file managers) that run without installation.
- 🏮 Mangas: Highly compressed, curated chapters for offline reading.
- 🎵 Music/Phonk: A selection of high-energy phonk and popular tracks optimized for file size.
- 🖼 Image Gallery: A collection of wallpapers and minimalist photography.
- 🤖 Offline AI: Lightweight LLMs (like TinyLlama) or basic AI scripts that run locally.
- 🔥 Survival Knowledge: First aid, navigation, and wilderness survival documentation.
- 📖 Dictionary: Full-text offline linguistic and technical references.
